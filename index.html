Weglot intern project

From august 2019 to January 2020 I was an intern at Weglot, a growing startup in Paris that created an app to automate website translations. In addition to the job of product specialist I had there I delivered two projects to help the team work faster.
The biggest one is the automation of the churn reporting process, it previously needed half an hour to one person each day to report each client cancellation in a google sheet, I created a python program to communicate with multiple APIs in order to gather all the needed information (Chart Mogul, helpscout, google, stripe) and then write them in the google sheet automatically. The program runs on an Ubuntu instance on aws, thanks to a CRON command each day at 3am the code runs itself. The most interesting part of it is the mail analysis, each cancellation is associated to a mail explaining the reason (to expensive, no need, ...) and the mails need to be sorted in six categories, I implemented a decision tree to automatically sort the mails, the only flaw of that method is that a lot of emails just make no sense, so sometimes human correction is needed.

Urban Farm

Urban Farm is a project we developed at ISEP by teams. Me and my team created a connected henhouse linked to a website to allow the farmer to remotely control everything.

Thanks to multiple sensors on an energia board connected by wifi we implementedd a lot of features to the henhouse:

  • The detection of a danger thanks to microphones and a program that detected chickens panic shouts due to a fox attack for example, this was linked to a powerful speaker to scare the predator.
  • Control of inside temperature thanks to temperature sensors and heaters.
  • Control of light thanks to a lamp and a light sensor.
  • Motion sensors allowed us to count the chicken entering or leaving the house and a motor could automatically close the door at a given time when ever chicken are inside.
  • The food delivery was also automated thanks to motors.

The energia board was linked to a website we developped to allow the user to control everything from home.

Not only this project allowed us to discover all digital technologies from electronic to signal processing and web development, but it also taught us how to work in a team.

Neural network library

In order to understand how neural networks work I implemented this simple python library. It can create a neural network calculate the propagation of an input and make the backpropagation in order to train the network and finally display the network with all weights and biases that it used. As an example I used it to make a neural network that behaves like a XOR logic gate. Check the project on github.

Double pendule simulator

I created a double pendule simulator in javascript, you can change the mass and length of both pendules as well as the initial angle and the gravittional constant and then launch the simulation. I used the Euler method to solve the position equation at each time period. The equation resolution is detailed in the projet's pdf.
